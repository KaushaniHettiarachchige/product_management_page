- A simple Product Management Page built using Next.js App Router and Tailwind CSS.
- This Application allows users to add, view, update and delete products.
- And here used local storage to store data without any backend and database.

## Features
- Add products
- View products
- Edit/Update products
- Delete products
- Search product by name
- Form validation
- Toast notification
- Dark Mode Toggle
- Image upload
- Local storage for store data

## Set up instructions
- Clone the repository:  git clone https://github.com/KaushaniHettiarachchige/product_management_page.git

- Navigate to project folder: cd product-management-app

- Install dependencies: npm install

- Run the development server: npm run dev

- Open the application on your browser: http://localhost:3000

## Tech stack used
- Next.js App Router
- React.js
- Tailwind CSS
- JavaScript
- For toast notifications - react-hot-toast
- For dark mode - next-themes
- For dark mode toggle - Radix UI Switch

## How the application can work
- Users can add product by entering product name, price, description and image of the product.
- Uploaded products are displayed as product cards.
- Each card includes product image, product name, price, description,
- Each product can update product details using edit button.
- Each product can remove from the list using delete button.
- Product can find by searching product name.
